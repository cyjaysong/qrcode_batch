# 批量生成二维码工具

## 功能说明
该工具以纯web的方式提供服务。
该工具用于批量生成二维码图片，支持从Excel文件中读取数据并生成对应的二维码。可自定义二维码内容、大小和样式。
可自由编辑布局，支持元素包含文本、二维码、图片三种类型，支持自定义样式。可调节元素大小，可设置二维码容错率、码版本、边距、码背景色等参数。文本和二维码内容支持选择Excel文件中的指定列进行生成

## 使用方法

1. 准备Excel文件，包含需要生成二维码的数据
2. 运行程序，选择Excel文件
3. 程序会自动生成对应的二维码图片

## 依赖环境

- Node.js v22.21.1 (已通过.nvmrc指定)
- pnpm包管理器

## 安装依赖

```bash
pnpm install
```

## 运行程序

```bash
pnpm run dev
```
